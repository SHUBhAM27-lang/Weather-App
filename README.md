# 🌦️ Weather App

A real-time weather application built using React.js and OpenWeather API.

## 🚀 Features

- Current weather data
- 5-day forecast
- Search any city
- Responsive UI

## 🛠️ Technologies Used

- React.js
- Tailwind CSS
- OpenWeather API
- Vite

## ▶️ How to Run

```bash
npm install
npm run dev
```
